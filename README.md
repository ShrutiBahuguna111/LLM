# LLM Learning Repository

This repository contains my hands-on practice and implementation of Large Language Model (LLM) concepts learned from Sebastian Raschka's **LLMs From Scratch** course.

## Topics Covered
1. Tokenization & Embeddings
2. Self-Attention & Multi-Head Attention
3. Transformer Architecture
4. GPT Model Implementation
5. Loading Pretrained Weights
6. Text Generation Techniques
7. LLM Pretraining Concepts
8. Instruction Fine-Tuning
9. Classification Fine-Tuning
10. Model Saving & Loading


## Purpose

The goal of this repository is to understand how modern LLMs work by implementing key components from scratch and experimenting with different stages of the LLM pipeline.
